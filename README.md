Base project setup with:
- Webpack

## Installation

```
yarn install
```

## Run development mode

```
yarn start
```

## Build production mode

```
yarn build
```

## Linting

The rules are in `.eslintrc.js` and in `.prettierrc.js`

To run the linter:
```
yarn lint
```

To run the linter and fix the autofixable errors:
```
yarn lint-fix
```
